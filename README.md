# Oasis Plant Store

Oasis Plant Store is an e-commerce website for purchasing indoor and decorative plants, including product listings, categories, flash sales, and personalized recommendations.

## Features

- Browse a variety of indoor and decorative plants
- View products by categories
- Flash sales and discounts
- Personalized recommendations
